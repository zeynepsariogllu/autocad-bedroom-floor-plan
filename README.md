# autocad-bedroom-floor-plan

## Overview

This project presents a 2D bedroom floor plan designed using AutoCAD.

The design includes room dimensions, furniture placement, and space organization principles. The objective was to create a functional bedroom layout while applying technical drawing and drafting techniques.

## Features

* 2D architectural floor plan
* Furniture placement and arrangement
* Room dimensioning
* Technical drafting standards
* Interior space planning

## Room Components

* Bed
* Wardrobe
* Nightstand
* Desk
* Chair

## Software Used

* AutoCAD

## Skills Demonstrated

* Technical Drawing
* Floor Plan Design
* Space Planning
* Dimensioning
* Interior Layout Design
